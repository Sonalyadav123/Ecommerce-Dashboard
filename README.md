#  E-Commerce Dashboard (Power BI)

##  Project Overview
This is an interactive Power BI dashboard built to analyze e-commerce data.

##  Features
- KPI Metrics (Sales, Orders, Customers, Rating)
- Sales Trend Analysis
- Category & City Insights
- Payment Method Analysis
- Customer Review Analysis

##  Tools Used
- Power BI
- Excel
- Data Cleaning
- DAX

##  Dashboard Preview
![Dashboard](Dashboard.png)

##  Key Insights
- High sales observed in top cities
- Majority users prefer digital payment methods
- Customer review analysis shows engagement trends
